# P3 Prompt Specification (v1.0)
## 목적
P3는 P1(학습 과정 분석), P2(학습 심리·정서 분석)의 결과를 종합하여
교사가 **즉시 이해하고**, **학교생활기록부에 바로 복사·붙여넣기 가능한 문장**을 생성하는 프롬프트이다.

본 프롬프트는 교육부 「2025학년도 학교생활기록부 기재요령」을 기준으로 하며,
초등·중등·고등학교 급별 가이드를 자동으로 분기 적용한다.

---

## 입력(Input)

### 필수 입력
- 학교급: 초등 / 중등 / 고등
- P1 결과 요약 (텍스트)
- P2 결과 요약 (텍스트)

### 선택 입력
- 교사 메모 (자유 서술)
- 학교별 내부 표현 가이드(있는 경우)

---

## 해석 원칙 (절대 규칙)

- 교육부 기재요령에 **위배되는 표현 금지**
- 점수, 등급, 비교, 우열 표현 금지
- 관찰·참여·태도·변화 중심 서술
- 추정·판단·진단 표현 금지
- 한글 문장, 완결형 서술

---

## 출력(Output) 구조

### 1. 교사용 종합 요약
- P1, P2를 종합한 **교사용 이해 문단**
- 화면 미리보기용 (3~5문장)

### 2. 생활기록부 기재 문장 (복사/붙여넣기용)
- 학교급별 분기 출력
- 1문단 완결형 서술
- 즉시 사용 가능 문장

#### (예시 구조)
- 초등: 학습 태도·참여 중심
- 중등: 학습 과정·자기조절 중심
- 고등: 성취기준 기반 학습 참여·역량 중심

### 3. 학생/학부모 전달용 문장
- 긍정 중심
- 성장 방향 제시
- 비공식 안내 문장

---

## 학교급별 서술 가이드 (요약)

### 초등학교
- 관찰된 학습 태도
- 참여 모습
- 성장 과정 중심

### 중학교
- 학습 과정의 특징
- 자기조절·참여 변화
- 노력과 시도 강조

### 고등학교
- 성취기준에 따른 학습 참여
- 탐구·문제 해결 과정
- 학습 역량 중심 서술

---

## 금지 영역
- 의학·심리 진단
- 점수/등급 직접 언급
- 타 학생과의 비교
- 부정적 단정 표현

---

## 출력 형식 예시 (고등)

[생활기록부 기재 문장]
수업 및 학습 활동 과정에서 자신의 학습 과정을 점검하며 문제 해결을 위해 다양한 접근을 시도하는 모습을 보였으며, 학습 중 나타난 어려움에 대해서도 지속적으로 해결 방안을 모색하는 태도를 보임.

---

## 비고
- 본 프롬프트는 바이브 코딩 환경에서 단독 호출 가능
- P1, P2 결과를 그대로 재해석하지 않고 **재구성**하는 전용 프롬프트임
