## 추가된 기능

### 1. 도구 실행 화면 레이아웃
PRD 문서에 따라 도구 타일 클릭 시 표시되는 실행 화면을 구현했습니다.

#### 레이아웃 구조
```
┌─────────────────────────────────────────────────────────────────┐
│  [← 뒤로] 퀴즈 생성기                           [저장 옵션 ▼]  │
├──────────────────────────┬──────────────────────────────────────┤
│                          │                                      │
│   왼쪽: 정보 입력          │   오른쪽: 미리보기                    │
│   - 과목 선택             │   - 생성된 산출물 표시                │
│   - 학년 선택             │   - 로딩 상태 표시                   │
│   - 주제 입력             │   - 빈 상태 안내                     │
│   - 난이도 선택           │                                      │
│   [생성하기]              │                                      │
│                          │                                      │
└──────────────────────────┴──────────────────────────────────────┘
```

### 2. 주요 기능

#### 2.1 헤더 영역
- **뒤로가기 버튼**: 도구 목록으로 돌아가기
- **도구 제목**: 현재 사용 중인 도구 표시
- **저장 옵션 드롭다운**:
  - ⬇️ 다운로드 (파일로 저장)
  - 📋 복사 (클립보드에 복사)
  - 📁 만든 자료함에 저장

#### 2.2 입력 영역 (왼쪽 40%)
- 도구별 커스텀 입력 폼
- 샘플 구현:
  - 과목 입력
  - 학년 선택 (드롭다운)
  - 주제 입력
  - 난이도 선택 (드롭다운)
  - 생성하기 버튼 (입력 완료 시 활성화)

#### 2.3 미리보기 영역 (오른쪽 60%)
- 생성된 산출물 실시간 표시
- 로딩 상태:
  - 스피너 애니메이션
  - "생성 중..." 메시지
- 빈 상태:
  - 안내 메시지 표시
  - "왼쪽에서 정보를 입력하고 '생성하기' 버튼을 클릭하세요."

### 3. 반응형 디자인

#### 데스크톱 (1024px 이상)
- 2단 레이아웃 유지
- 왼쪽 40% (2/5) + 오른쪽 60% (3/5)

#### 태블릿 (768px - 1023px)
- 2단 레이아웃 유지
- 왼쪽 50% + 오른쪽 50%

#### 모바일 (767px 이하)
- 1단 레이아웃으로 전환
- 입력 영역 → 미리보기 영역 순서로 세로 배치

## 사용 방법

### 1. 홈에서 도구 실행
1. 홈 페이지 접속 (http://localhost:5173)
2. 추천 도구 중 하나 클릭
3. 도구 실행 화면으로 이동
4. 왼쪽에 정보 입력
5. "생성하기" 버튼 클릭
6. 오른쪽에서 결과 확인
7. 오른쪽 상단에서 저장 옵션 선택

### 2. 도구 페이지에서 실행
1. 도구 페이지 접속 (http://localhost:5173/tools)
2. 검색/필터링으로 원하는 도구 찾기
3. 도구 타일 클릭
4. 동일한 실행 흐름
